# 学习单词组件状态机
不变量：一个用户只可以有一个学习会话

向后端请求学习会话是否存在，查询是否首次学习
- 是：从后端获取单词列表，初始化队列，获取单词详情
- 否：
    - 并行运行：
        - 查询单词列表缓存 -> 判断有没有
            - 如果有：加载单词列表缓存
            - 如果没有：从后端加载单词列表

        - 查询学习队列缓存 -> 判断有没有
            - 有：准备好队列标识
            - 没有：后端获取学习会话

    - 将缓存/加载结果设置为状态

    - 注入队列

    - 从后端查询当前展示单词的详细信息
