# 注册与登录测试用例清单

## 1. 文档目的

本清单用于对 `recite-word` 项目的注册和登录功能进行可复用测试，覆盖：

- 正常功能流程
- 表单校验
- 安全输入（SQL 注入、XSS 注入）
- 会话与访问控制
- 异常处理

## 4. 注册功能测试用例

| 用例ID | 用例名称 | 前置条件 | 测试步骤 | 预期结果 |
|---|---|---|---|---|
| REG-001 | 注册页打开 | 未登录 | 1) 访问 `/register` | 页面展示 Register 表单，包含 username、email、password、Register 按钮 |
| REG-002 | 必填校验-全部为空 | 未登录 | 1) 访问 `/register` 2) 不输入任何字段 3) 点击 Register | 用户名/密码出现必填提示；不应创建用户 |
| REG-003 | 必填校验-仅用户名为空 | 未登录 | 1) 输入合法 email、password 2) username 留空 3) 点击 Register | username 出现必填提示；不应创建用户 |
| REG-004 | 邮箱格式校验 | 未登录 | 1) 输入合法 username、password 2) email 输入 `not-an-email` 3) 点击 Register | 出现邮箱格式错误提示；不应创建用户 |
| REG-005 | 用户名字符集校验 | 未登录 | 1) email/password 合法 2) username 输入 `normal_user` 或 `a!b` 3) 点击 Register | 出现用户名字符限制提示；不应创建用户 |
| REG-006 | 密码字符集校验 | 未登录 | 1) username/email 合法 2) password 输入 `Passw0rd!` 3) 点击 Register | 出现密码字符限制提示；不应创建用户 |
| REG-007 | 密码长度校验 | 未登录 | 1) username/email 合法 2) password 输入过短值（如 `abc12`） 3) 点击 Register | 出现密码长度错误提示；不应创建用户 |
| REG-008 | 重复用户名/邮箱注册 | 已存在用户 `test20260329a` | 1) 用已注册用户名或邮箱再次注册 | 注册失败并提示冲突/已存在信息；系统不新增重复用户 |
| REG-009 | 注册成功-标准路径 | 未登录，账号未注册 | 1) 输入合法 username/email/password 2) 点击 Register | 显示注册成功页面或提示；可进入登录流程 |
| REG-010 | 协议勾选行为（若可操作） | 未登录 | 1) 取消协议勾选 2) 点击 Register | 应阻止提交并提示需勾选协议 |

## 5. 登录功能测试用例

| 用例ID | 用例名称 | 前置条件 | 测试步骤 | 预期结果 |
|---|---|---|---|---|
| LOG-001 | 登录页打开 | 未登录 | 1) 访问 `/login` | 页面展示 login 表单，包含 username、password、Log in 按钮 |
| LOG-002 | 必填校验-全部为空 | 未登录 | 1) 不输入字段 2) 点击 Log in | username/password 出现必填提示；不应登录成功 |
| LOG-003 | 用户名字符集校验 | 未登录 | 1) username 输入 ` ' OR '1'='1 ` 2) password 任意 3) 点击 Log in | 出现用户名格式/字符限制提示；不应登录成功 |
| LOG-004 | 密码字符集校验 | 未登录 | 1) username 合法 2) password 输入 `<script>alert(1)</script>` 3) 点击 Log in | 出现密码格式/字符限制提示；不应登录成功 |
| LOG-005 | 错误密码登录 | 已存在用户 | 1) 输入合法用户名 2) 输入错误密码 3) 点击 Log in | 登录失败，提示凭据错误（如 invalid credentials），HTTP 401 |
| LOG-006 | 不存在用户登录 | 未登录 | 1) 输入不存在用户名与任意密码 2) 点击 Log in | 登录失败，提示用户不存在或凭据错误 |
| LOG-007 | 登录成功-标准路径 | 已存在合法账号，未登录 | 1) 输入正确 username/password 2) 点击 Log in | 登录成功并跳转首页；导航显示登录态菜单 |
| LOG-008 | 已登录访问登录页拦截 | 已登录 | 1) 访问 `/login` | 页面提示已登录并阻止重复登录 |
| LOG-009 | 退出登录 | 已登录 | 1) 点击用户菜单 2) 选择 Logout | 成功退出，导航恢复未登录态 |
